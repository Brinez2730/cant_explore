Como saber se você esta sendo explorado

Siga esse fluxograma para saber se você esta sendo explorado socialmente ou não. Mas lembre-se, cada caso é particular, então faça uma analise mas considere todos os fatores possíveis.


Passo 1: Condições de Trabalho ou Vida

Você trabalha/vive em condições degradantes (ex.: jornada exaustiva, ambiente insalubre, moradia precária)?

Se sim: Avançe para 2.

Se não: Pode não ser exploração (avalie outros fatores).



Passo 2: Remuneração ou Benefícios

Recebe menos que o mínimo legal/justo ou não é remunerada?

Se sim: Avançe para 3.

Se não: Pode ser desvantagem, mas não necessariamente exploração.



Passo 3: Liberdade de Escolha

A pessoa pode deixar o trabalho/situação sem sofrer ameaças, dívidas ou violência?

Se sim: Indício de exploração, avançe para 4.

Se não: Pode ser má condição, mas com autonomia.



Passo 4: Controle e Coerção

Há documentos retidos, como RG e carteira de trabalho, dívida manipulada, como por exemplo servidão por dívida, ou ameaças psícologicas ou físicas?

Se sim: Exploração confirmada. Busque ajuda jurídica ou denuncie.

Se não: Pode ser caso de vulnerabilidade (avalie apoio social).
